# Phase 1 완벽 실행 계획서

## 🎯 **목표**: 30초 고품질 한국어 영상 완벽 구현

### **📋 Phase 1 상세 태스크 리스트**

#### **Task 1: 자막 동기화 시스템 완전 수정** 🚨 최우선
**현재 문제**:
- 2,3,4번째 자막 타이밍이 모두 00:00:00,000으로 표시
- 첫 번째 자막만 1.83초로 정상

**해결 방안**:
1. Whisper 자막 생성 로직 분석
2. SRT 파일 생성 과정 디버깅
3. 오디오-자막 동기화 알고리즘 수정
4. 타임스탬프 정확도 검증

**성공 기준**:
- 모든 자막이 정확한 타이밍을 가짐
- 음성과 자막이 완벽히 동기화
- SRT 파일 포맷이 올바름

---

#### **Task 2: 한국어 음성 시스템 실제 적용 검증** 🇰🇷
**현재 문제**:
- config.toml에서 ko-KR-SunHiNeural로 변경했으나
- script.json에서 여전히 de-AT-IngridNeural-Female 사용

**해결 방안**:
1. 음성 설정 로딩 로직 확인
2. UI에서 config.toml 반영 검증
3. 실제 TTS 호출 시 적용 확인
4. Azure TTS API 한국어 음성 테스트

**성공 기준**:
- 한국어 음성으로 나레이션 생성 확인
- 음성 품질이 자연스러움
- 발음이 정확함

---

#### **Task 3: 영상 길이 30초 연장 시스템 구현** ⏰
**현재 문제**:
- max_clip_duration이 15초로 설정되어 있음
- 실제 클립들이 3초씩만 사용됨
- 총 영상 길이가 너무 짧음

**해결 방안**:
1. max_clip_duration을 30초로 설정
2. 클립 수집 로직 개선 (더 많은 클립 확보)
3. 클립 반복/루핑 시스템 개선
4. 오디오 길이에 맞춘 영상 길이 조정

**성공 기준**:
- 30초 이상의 영상 생성
- 클립 간 자연스러운 연결
- 오디오와 영상 길이 완벽 매칭

---

#### **Task 4: 한글 폰트 완벽 적용 및 검증** 🔤
**현재 문제**:
- Charm-Bold.ttf로 설정했으나 실제 적용 확인 필요
- script.json에서 여전히 MicrosoftYaHeiBold.ttc 표시

**해결 방안**:
1. 폰트 로딩 시스템 확인
2. 한글 자막 렌더링 테스트
3. 폰트 파일 경로 및 존재 여부 확인
4. 자막 가독성 최적화

**성공 기준**:
- 한글 자막이 선명하게 표시
- Charm-Bold.ttf 폰트 정상 적용
- 자막 크기와 위치 최적화

---

#### **Task 5: 품질 검증 및 테스트 영상 생성** ✅
**검증 항목**:
1. 30초 영상 길이 확인
2. 한국어 음성 완벽 작동
3. 자막 동기화 정확성
4. 한글 폰트 적용
5. 클립 품질 및 연결성

**테스트 시나리오**:
- 주제: "2025년 AI 트렌드"
- 예상 길이: 30초
- 검증 포인트: 모든 기능 완벽 작동
