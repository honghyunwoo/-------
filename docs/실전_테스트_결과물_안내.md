# 실전 테스트 결과물 안내

실전 테스트를 통해 생성된 파일들의 위치와 확인 방법을 안내합니다.

## 📂 생성된 파일 위치

### 1. 썸네일 파일 (3개)
**위치**: `storage/test_thumbnails/`

```
storage/test_thumbnails/
├── thumbnail_curious.png      (1280x720, 24.2KB)
├── thumbnail_excited.png      (1280x720, 23.8KB)
└── thumbnail_surprised.png    (1280x720, 21.4KB)
```

**특징**:
- 유튜브 표준 해상도 (1280x720)
- 3가지 감정별 버전 (curious, excited, surprised)
- PNG 포맷, 평균 23KB
- 제목: "AI 영상 제작 자동화 완벽 가이드"

### 2. 자막 파일 (2개)
**위치**: `storage/test_subtitles/`

```
storage/test_subtitles/
├── test_subtitle.srt          (487 bytes)
└── test_subtitle.ass          (4,005 bytes)
```

**특징**:
- SRT: 표준 자막 포맷 (5개 세그먼트)
- ASS: MrBeast 스타일 고급 자막 (애니메이션 효과 포함)

## 📝 자막 파일 확인 방법

### SRT 자막 (test_subtitle.srt)
**내용 예시**:
```srt
1
00:00:00,000 --> 00:00:03,000
AI 영상 제작 자동화의 숨겨진 비밀을 공개합니다

2
00:00:03,000 --> 00:00:06,000
이것만 알면 누구나 바이럴 영상을 만들 수 있습니다

3
00:00:06,000 --> 00:00:09,000
첫 번째, 강력한 후킹으로 시작하세요
```

**확인 방법**:
- 텍스트 에디터로 열기 (VS Code, 메모장 등)
- 영상 편집 프로그램에서 불러오기 (Premiere, DaVinci Resolve 등)

### ASS 자막 (test_subtitle.ass)
**특징**:
- MrBeast 스타일 고급 자막
- 단어별 애니메이션 효과
- 색상 변화 및 크기 변화
- Karaoke 효과 (단어별 강조)

**ASS 파일 구조**:
```
[Script Info]        # 스크립트 정보
[V4+ Styles]         # 스타일 정의
  - standard: 기본 스타일 (80pt, 흰색)
  - dramatic: 극적 스타일 (100pt, 금색)
  - energetic: 활기찬 스타일 (90pt, 빨강)
  - calm: 차분한 스타일 (70pt, 베이지)
[Events]             # 자막 이벤트 (단어별 타이밍)
```

**확인 방법**:
- Aegisub (자막 편집 전문 프로그램)
- VLC 플레이어로 영상과 함께 재생
- 텍스트 에디터로 코드 확인

## 🎬 영상에 적용하는 방법

### 1. 썸네일 사용
```python
from app.services.thumbnail_generator import ThumbnailGenerator

generator = ThumbnailGenerator()
thumbnail = generator.generate_viral_thumbnail(
    title="내 영상 제목",
    keywords=["키워드1", "키워드2"],
    emotion="excited"
)

# 파일로 저장
thumbnail.save("my_thumbnail.png")
```

**유튜브 업로드 시**:
1. 영상 업로드 페이지에서 "맞춤 미리보기 이미지" 클릭
2. 생성한 썸네일 파일 선택
3. 자동으로 1280x720 크기로 최적화되어 적용됨

### 2. 자막 사용

#### Premiere Pro
1. 프로젝트에 자막 트랙 추가
2. SRT 또는 ASS 파일 import
3. 타임라인에 배치

#### DaVinci Resolve
1. Edit 페이지 → Subtitle 트랙
2. Import Subtitle → ASS 파일 선택
3. 스타일 자동 적용

#### FFmpeg (명령줄)
```bash
ffmpeg -i input.mp4 -vf "ass=test_subtitle.ass" output.mp4
```

1. **저작권**: 생성된 이미지는 자유롭게 사용 가능
2. **파일 크기**: 썸네일은 2MB 이하 권장 (현재 평균 23KB ✅)
3. **해상도**: 유튜브는 1280x720 이상 권장 (현재 정확히 1280x720 ✅)
4. **자막 인코딩**: UTF-8 인코딩 사용 (한글 완벽 지원 ✅)

## 🆘 문제 해결

### 썸네일이 보이지 않는 경우
```bash
# 파일 존재 확인
ls storage/test_thumbnails/

# 권한 확인
chmod 755 storage/test_thumbnails/
```

### 자막이 깨지는 경우
- UTF-8 인코딩 확인
- BOM 없는 UTF-8 사용
- Windows 메모장 대신 VS Code 사용 권장
