## 🎯 프로젝트 목표

MoneyPrinterTurbo 오픈소스를 기반으로 **한국어 특화 AI 영상 제작 플랫폼**을 구축합니다.

### 핵심 목표
1. **한국 시장 특화**: 한글 폰트, 한국어 TTS, 한국 문화에 맞는 콘텐츠
2. **바이럴 콘텐츠 생성**: YouTube 트렌드 기반 자동 콘텐츠 제작
3. **고품질 영상**: 4K/Full HD 소스, 8000k 비트레이트 인코딩
4. **완전 자동화**: 스크립트 생성부터 썸네일까지 원클릭 생성

## 🏗️ 시스템 아키텍처

### 기술 스택

**백엔드**
- Python 3.13.7
- FastAPI (REST API)
- Streamlit (Web UI)
- PostgreSQL (데이터베이스)

**AI/ML**
- OpenAI GPT-4 (스크립트 생성)
- Azure TTS (음성 합성)
- Whisper Large-v3 (자막 생성)
- PIL/Pillow (썸네일 생성)

**비디오 처리**
- MoviePy (영상 편집)
- FFmpeg (인코딩)
- ImageMagick (이미지 처리)

**외부 API**
- Pexels API (스톡 영상)
- Pixabay API (스톡 영상)
- YouTube Data API v3 (트렌드 분석)

### 디렉토리 구조

```
MoneyPrinterTurbo/
├── app/                    # 핵심 애플리케이션
│   ├── config/            # 설정 관리
│   ├── models/            # 데이터 모델 (Pydantic)
│   ├── services/          # 비즈니스 로직
│   │   ├── llm.py        # LLM 스크립트 생성
│   │   ├── voice.py      # TTS 음성 생성
│   │   ├── material.py   # 영상 소재 검색/다운로드
│   │   ├── video.py      # 영상 편집/합성
│   │   ├── subtitle.py   # 자막 생성
│   │   ├── thumbnail_generator.py  # 썸네일 생성
│   │   ├── viral_hooks.py          # 바이럴 훅 생성
│   │   ├── youtube_analyzer.py     # YouTube 트렌드 분석
│   │   └── seo_optimizer.py        # SEO 최적화
│   └── utils/             # 유틸리티 함수
├── webui/                 # Streamlit 웹 인터페이스
├── storage/               # 생성된 파일 저장
│   ├── cache_videos/     # 다운로드한 영상
│   ├── tasks/            # 작업별 결과물
│   └── test_*/           # 테스트 결과물
├── docs/                  # 프로젝트 문서
├── tests/                 # 테스트 코드
└── config.toml           # 시스템 설정

```

## 🚀 주요 기능

- ✅ 영상 소재 검색/다운로드 (Pexels/Pixabay)

- ✅ 바이럴 훅 생성 (10가지 스타일)

- ✅ SEO 최적화 (제목/설명/태그)
- ✅ 고급 자막 스타일 (MrBeast 스타일)

### Phase 3: 품질 개선 (진행 중)

- ✅ 한글 폰트 지원 (영상 자막)
- ✅ 고품질 인코딩 (8000k bitrate, slow preset)
- ✅ 고해상도 소스 선택 (4K/Full HD 우선)
- ⏳ 풀 파이프라인 테스트

- ⏳ 구독 결제 시스템

- ⏳ 브랜딩 커스터마이징
