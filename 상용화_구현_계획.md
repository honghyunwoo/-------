# 🦉 올빼미 AI 영상 스튜디오 상용화 구현 계획

## 📋 Phase 1: 기본 상용화 시스템 (2-3주)

### 1. 사용자 인증 시스템 구현
```python
# 새로 추가할 파일들:
- app/models/user.py          # 사용자 모델
- app/services/auth.py        # 인증 서비스
- app/controllers/auth.py     # 인증 컨트롤러
- app/middleware/auth.py      # 인증 미들웨어
```

**구현 내용:**
- JWT 토큰 기반 인증
- 회원가입/로그인 API
- 소셜 로그인 (구글, 카카오)
- 비밀번호 재설정 기능

### 2. 데이터베이스 시스템 구축
```python
# 새로 추가할 파일들:
- app/database/connection.py  # DB 연결
- app/database/models.py      # SQLAlchemy 모델
- app/database/migrations/    # 마이그레이션
```

**구현 내용:**
- PostgreSQL 데이터베이스
- 사용자 테이블
- 구독/결제 테이블
- 영상 생성 히스토리 테이블

### 3. 결제 시스템 연동
```python
# 새로 추가할 파일들:
- app/services/payment.py     # 결제 서비스
- app/controllers/payment.py  # 결제 컨트롤러
- app/models/subscription.py  # 구독 모델
```

**구현 내용:**
- 아임포트/토스페이먼츠 API 연동
- 구독제 모델 (베이직/프로/비즈니스)
- 크레딧 시스템
- 결제 웹훅 처리

### 4. 사용량 제한 시스템
```python
# 새로 추가할 파일들:
- app/services/usage.py       # 사용량 관리
- app/middleware/rate_limit.py # 레이트 리미팅
```

**구현 내용:**
- 사용자별 월간 영상 생성 제한
- API 호출 제한
- 실시간 사용량 추적

## 🚀 즉시 실행 가능한 작업들

### 1. 환경 설정
```bash
# 필요한 패키지 설치
pip install sqlalchemy alembic psycopg2-binary
pip install python-jose[cryptography] passlib[bcrypt]
pip install python-multipart
pip install stripe  # 또는 아임포트 SDK
```

### 2. 데이터베이스 설정
```python
# config.toml에 추가
[database]
url = "postgresql://user:password@localhost/owl_studio"
echo = false
```

### 3. 인증 설정
```python
# config.toml에 추가
[auth]
secret_key = "your-secret-key-here"
algorithm = "HS256"
access_token_expire_minutes = 30
```
