# Script V2.1 - 대본 작성 가이드
> 패널 합의안(V2) + 완주율 개선 2규칙(V2.1) (2026-01-06)

## 핵심 스펙
- 총 길이: 30-40초
- 총 글자수(6문장 합): 225-285자
- 문장 수: 6문장 고정 (S1~S6)
- 톤앤매너: "먼저 넘어진 동료"형
- 말투: 반말, 단문, 구어체
- broll_keywords: 영어 3개 고정 (사물/상황만)

## 운영 규칙 (필수)

### 규칙 1) 문장당 1아이디어
- S2/S3/S4는 설명/비유/감정 중 하나만 선택한다.
- 한 문장에 "설명+비유+감정" 섞지 않는다.

### 규칙 2) 자막 밀도 상한
- 한 문장 2줄 이내(권장 50자, 최대 55자)
- 길면 문장 안에서 군더더기 삭제(6문장 구조는 유지)

### 규칙 3) S5 Action은 제안형 통일
- 명령형(~해라/하라/꺼라) 금지
- 제안형(~해보자/~해볼래)만 사용한다.

### 규칙 4) 루프 CTA 3종 로테이션
- 인정형: "또 왔다는 건 아직 포기 안 했다는 거야."
- 축하형: "지금 여기까지 온 게 이미 한 번 이긴 거야."
- 연결형: "다음 편에서 한 단계만 더 줄여보자."
(매번 같은 패턴 금지, 3종을 번갈아 쓴다.)

## V2.1 추가 규칙 (완주율 개선용)

### 규칙 5) 리텐션 브릿지 1회 강제
- S3 끝 또는 S4 시작에 "한 박자 끌기"를 1회 넣는다.
- 예시:
  - "여기서 갈린다."
  - "근데 진짜는 지금부터야."
  - "포인트가 하나 있어."
  - "여기가 핵심이야."

### 규칙 6) S5에 '마찰 제거' 1개 포함
- 행동(2분 이내) + "실패 부담 제거"를 같이 적는다.
- 예시:
  - "버튼만 누르고 멈춰도 돼."
  - "옮기기만 해도 돼."
  - "10초만 해도 충분해."
  - "한 번만 해도 괜찮아."

## 금지 표현 목록

### 장면/화면 지칭 금지
- "지금 화면", "보이시죠", "여기 보면", "이 장면", "화면에", "영상에서"
- (※ '스크롤'은 행태 설명이므로 허용)

### 의료 표현 금지
- "치료", "완치", "진단", "병원", "우울증", "불안장애", "~에 좋다", "건강해진다"

### 재무 조언/확정 표현 금지
- "투자", "종목", "수익보장", "수익률", "100% 확정", "경제적 자유", "부자 된다"
- (※ 소비 트리거: 월급날/카드값/구독/배달앱/장바구니는 허용)

### 과장/클리셰 금지

- "노력하면 된다", "포기하지 마", "할 수 있다", "꿈은 이루어진다"
- "여러분", "안녕하세요", "오늘은"

## Validator 체크리스트

1. 총 글자수(6문장 합): 225-285자
2. 각 문장 길이: ≤ 55자
3. 금지어 패턴 체크
4. 명령형(~해라/하라/꺼라) 금지
5. broll_keywords: 정확히 3개, 영어만
6. cta_type: "인정형" | "축하형" | "연결형"
7. **S3에 브릿지 키워드 포함** (여기서/갈린다/진짜는/핵심)
8. **S5에 마찰제거 키워드 포함** (돼/충분/괜찮/해도)
