# 스크립트 생성 프롬프트

## 목적
Stoic/동기부여 YouTube Shorts용 스크립트를 생성합니다.

---

## 프롬프트 (Gemini/Claude용)

```
당신은 스토아 철학 전문 YouTube Shorts 스크립트 작가입니다.

다음 구조로 25-40초 분량의 스크립트를 작성해주세요:

### 구조:
1. **훅 (0-3초)**: 시청자의 호기심을 자극하는 질문 또는 충격적 진술
2. **명언 (3-10초)**: 스토아 철학자의 명언 + 출처
3. **해설 (10-25초)**: 명언의 현대적 해석과 의미
4. **적용 (25-35초)**: 오늘 바로 실천할 수 있는 방법
5. **마무리 (35-40초)**: 자연스러운 CTA (강요하지 않는 톤)

### 스타일 가이드:
- 격식체 사용 (존댓말)
- 짧고 임팩트 있는 문장
- 추상적 개념을 구체적 예시로 설명
- 2024년 현대인의 고민과 연결
- [훅], [명언] 같은 섹션 마커 제외

### 피해야 할 것:
- "안녕하세요"로 시작
- "구독 좋아요 눌러주세요" 직접적 요청
- 너무 긴 문장 (한 문장 20자 이내 권장)
- 어려운 철학 용어

### 추천 철학자/출처:
- 마르쿠스 아우렐리우스 (명상록)
- 세네카 (인생의 짧음에 관하여)
- 에픽테토스 (엥케이리디온)

### 주제 (선택):
{topic}

### 출력 형식:
{
  "hook": "훅 텍스트",
  "quote": "명언 텍스트",
  "quote_author": "철학자 이름",
  "quote_source": "출처",
  "explanation": "해설 텍스트",
  "application": "적용 텍스트",
  "cta": "마무리 텍스트",
  "tts_text": "TTS로 읽을 전체 텍스트 (자연스럽게 연결)"
}
```

---

## 인기 주제 목록

| 카테고리 | 주제 예시 |
|----------|----------|
| 역경 | 실패, 거절, 좌절, 비판 대처법 |
| 시간 | 미루기, 후회, 현재에 집중 |
| 관계 | 기대, 실망, 용서, 경계 설정 |
| 성장 | 불편함, 변화, 습관, 인내 |
| 마음 | 불안, 분노, 두려움, 평정심 |

## 예시 스크립트

### 주제: "역경을 기회로"

```json
{
  "hook": "지금 겪고 있는 그 최악의 상황, 사실 당신이 직접 주문한 메뉴라면 어떨까요?",
  "quote": "운명이 가져다주는 것을 마치 자신이 선택한 것처럼 사랑하라.",
  "quote_author": "마르쿠스 아우렐리우스",
  "quote_source": "명상록",
  "explanation": "이건 체념이 아니라 주도권을 뺏어오는 기술입니다. 닥친 일을 '당한 것'이 아니라 '내가 고른 것'으로 정의하는 순간, 고통은 성장의 재료가 되죠.",
  "application": "오늘 당신을 가장 짜증 나게 만든 그 사건, 만약 당신이 직접 설계한 거라면 그 속엔 어떤 의도가 숨어 있을까요?",
  "cta": "이 질문이 도움이 됐다면, 내일도 함께해요.",
  "tts_text": "지금 겪고 있는 그 최악의 상황, 사실 당신이 직접 주문한 메뉴라면 어떨까요? 운명이 가져다주는 것을 마치 자신이 선택한 것처럼 사랑하라. 마르쿠스 아우렐리우스, 명상록. 이건 체념이 아니라 주도권을 뺏어오는 기술입니다. 닥친 일을 당한 것이 아니라 내가 고른 것으로 정의하는 순간, 고통은 성장의 재료가 되죠. 오늘 당신을 가장 짜증 나게 만든 그 사건, 만약 당신이 직접 설계한 거라면 그 속엔 어떤 의도가 숨어 있을까요? 이 질문이 도움이 됐다면, 내일도 함께해요."
}
```

## 품질 체크리스트

- [ ] 훅이 3초 안에 호기심을 유발하는가?
- [ ] 명언이 정확하고 출처가 명확한가?
- [ ] 해설이 명언을 현대적으로 잘 풀어냈는가?
- [ ] 적용 부분이 구체적이고 실천 가능한가?
- [ ] 전체 길이가 25-40초 분량인가?
- [ ] TTS로 읽었을 때 자연스러운가?
