## 2. 영상 생성 SOP

### 2.1 파이프라인 실행 절차

```bash
# Step 1: 작업 디렉토리 이동
cd /home/user/shorts-factory

# Step 2: 가상환경 활성화
source venv/bin/activate

# Step 3: 명언 확인 (사용 안 한 명언)
python main.py list-unused --limit 5

# Step 4: 영상 생성
python main.py generate --quote-id 15 --hook H1 --cta C3 --channel stoic

# Step 5: 결과 확인
ls -la output/$(date +%Y-%m-%d)*/
```

## 5. 트러블슈팅

### 5.1 파이프라인 오류

| 오류 | 원인 | 해결 |
|-----|-----|-----|
| Claude API 오류 | API 키/할당량 | 키 확인, 재시도 |
| TTS 실패 | TYPECAST 서버 | 재시도, Google TTS 백업 |
| 영상 합성 실패 | 메모리/디스크 | 재부팅, 공간 확보 |
| 자막 타이밍 오류 | 오디오 길이 불일치 | 수동 조정 |

### 5.2 수동 복구 절차

```bash
# TTS 수동 생성 (TYPECAST 실패 시)
python core/tts_engine.py --text "스크립트 내용" --output audio.mp3

# 자막 수동 생성
python core/subtitle_generator.py --audio audio.mp3 --output captions.srt

# 영상 수동 합성
python core/video_composer.py \
  --audio audio.mp3 \
  --srt captions.srt \
  --broll assets/b-roll/nature/ \
  --output video.mp4
```
