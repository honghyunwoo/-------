### 2.2 산출물 정의

#### 채널별 산출물

| 구분 | 채널 1: 스토아 철학 | 채널 2: 영어학습 |

| 채널명 | 매일 스토아 | (미정) |

| 업로드 빈도 | 일 1회 → 주 5회 | 일 1회 → 주 5회 |
| 언어 | 한국어 | 한국어 + 영어 |
| TTS | TYPECAST | ElevenLabs |

#### 영상 패키지 산출물

```
/output/{날짜}_{제목}/
├── video.mp4        # 최종 영상 (1080x1920, 30fps)

├── captions.srt     # 자막 파일
├── thumbnail.png    # 썸네일 (선택)
└── metadata.json    # 제목, 설명, 태그
```

## 6. HOW (어떻게)

### 6.1 기술적 접근

```
┌─────────────────────────────────────────────────────────────┐
│                      기술적 접근                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  📌 자청 방법론 적용                                        │
│  ├── 패턴화: 훅/본문/CTA 템플릿화                          │
│  ├── 공식화: 성공 영상 공식 추출                           │
│  ├── 재검토 루프: AI 자체 평가 + 개선                      │
│  └── 예시 기반: 프롬프트에 성공 예시 포함                  │
│                                                             │
│  📌 기존 코드 활용                                          │
│  ├── video.py → 영상 합성 엔진                             │
│  ├── subtitle.py → 자막 생성                               │
│  └── 불필요 부분(DB, 인증) 제거                            │
│                                                             │
│  📌 외부 API 활용                                           │
│  ├── Claude: 스크립트 생성 + 재검토                        │
│  ├── TYPECAST: 한국어 TTS                                  │
│  └── ElevenLabs: 영어 TTS                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 프로세스 흐름

```
═══════════════════════════════════════════════════════════════════
                         HOW: 프로세스 흐름
═══════════════════════════════════════════════════════════════════

1️⃣ 기획 (1회성)
───────────────────────────────────────────────────────────────────
   명언 라이브러리 구축 → 템플릿 작성 → 프롬프트 설계

2️⃣ 생성 (매일)
───────────────────────────────────────────────────────────────────
   명언 선택 → 스크립트 생성 → 재검토 → TTS → 영상 합성

3️⃣ 검토 (매일)
───────────────────────────────────────────────────────────────────
   스크립트 확인 (1분) → 영상 확인 (1분) → 수정 또는 승인

4️⃣ 업로드 (매일)
───────────────────────────────────────────────────────────────────
   YouTube Studio → 제목/설명 입력 → 예약 업로드

5️⃣ 분석 (주간)
───────────────────────────────────────────────────────────────────
   성과 확인 → 패턴 분석 → 공식 개선

═══════════════════════════════════════════════════════════════════
```
