# PRD (Product Requirements Document)
# 유튜브 쇼츠 자동화 시스템 - Shorts Factory

---

## 1. 개요 (Overview)

### 1.1 제품명
**Shorts Factory** (쇼츠 팩토리)

### 1.2 제품 설명
유튜브 쇼츠 콘텐츠를 80% 자동화하여 생성하는 개인용 파이프라인 시스템.
두 개의 채널(스토아 철학 동기부여 / 영어학습)을 운영하며,
스크립트 생성부터 영상 합성까지 자동화하고 최종 검토 및 업로드만 수동으로 진행.

### 1.3 문서 버전
| 버전 | 날짜 | 작성자 | 변경 내용 |
|-----|------|-------|----------|
| 1.0 | 2026-01-03 | Claude | 초안 작성 |

---

## 2. 목표 및 배경 (Goals & Background)

### 2.1 프로젝트 배경
- 유튜브 쇼츠의 급성장 (일 700억 뷰)
- AI 도구의 발전으로 콘텐츠 자동화 가능
- 자청 방법론(패턴화/공식화)의 영상 제작 적용
- 기존 올빼미 AI 영상 스튜디오 코드 자산 활용

### 2.2 비즈니스 목표
| 목표 | 지표 | 기간 |
|-----|-----|-----|
| 채널 런칭 | 2개 채널 개설 | 1주 |
| 초기 성장 | 각 채널 500 구독자 | 2개월 |
| YPP 가입 | 광고 수익화 자격 | 3개월 |
| 자동화 완성 | 영상당 10분 이내 제작 | 1개월 |

### 2.3 사용자 목표
- 최소한의 시간 투자로 유튜브 채널 운영
- 콘텐츠 품질 유지하면서 양산 가능
- 두 개의 니치에서 동시 성장

## 4. 기능 요구사항 (Functional Requirements)

### 4.1 핵심 기능 (Must Have)

#### F1: 스크립트 자동 생성
| ID | F1 |
|-----|-----|
| 설명 | 입력된 주제/명언을 기반으로 쇼츠 스크립트 자동 생성 |
| 입력 | 명언/주제, 훅 유형, CTA 유형 |
| 출력 | 구조화된 스크립트 (훅→본문→CTA) |
| 기술 | Claude API |
| 우선순위 | P0 (필수) |

#### F2: 재검토 루프
| ID | F2 |
|-----|-----|
| 설명 | 생성된 스크립트를 AI가 자체 평가하고 개선 |
| 입력 | 초안 스크립트 |
| 출력 | 개선된 스크립트 (8점 이상) |
| 기술 | Claude API |
| 우선순위 | P0 (필수) |

#### F3: TTS 음성 생성
| ID | F3 |
|-----|-----|
| 설명 | 스크립트를 자연스러운 음성으로 변환 |
| 입력 | 텍스트 스크립트 |
| 출력 | MP3 오디오 파일 |
| 기술 | TYPECAST (한국어), ElevenLabs (영어) |
| 우선순위 | P0 (필수) |

#### F4: 영상 합성
| ID | F4 |
|-----|-----|
| 설명 | 오디오 + B-roll + 자막을 합성하여 최종 영상 생성 |
| 입력 | 오디오, 스크립트, 테마 |
| 출력 | MP4 영상 (9:16 비율) |
| 기술 | MoviePy |
| 우선순위 | P0 (필수) |

#### F5: 자막 생성 및 번인
| ID | F5 |
|-----|-----|
| 설명 | 오디오에서 자막 추출 또는 스크립트 기반 자막 생성 |
| 입력 | 오디오 또는 스크립트 |
| 출력 | SRT 파일 + 영상 내 자막 |
| 기술 | Whisper, MoviePy |
| 우선순위 | P0 (필수) |

#### F6: 메타데이터 생성
| ID | F6 |
|-----|-----|
| 설명 | 영상 제목, 설명, 태그, 해시태그 자동 생성 |
| 입력 | 스크립트, 채널 유형 |
| 출력 | metadata.json |
| 기술 | Claude API 또는 템플릿 |
| 우선순위 | P1 (높음) |
